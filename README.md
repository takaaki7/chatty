# Chatty
Repository of Facebook bot Chat App [Chatty](https://www.facebook.com/ChattyMessengerBot/)(service has been suspended )

### Get started

##### Setup wercker.
http://devcenter.wercker.com/docs/cli/installation

##### Run 

```
wercker dev --publish 8080
```
